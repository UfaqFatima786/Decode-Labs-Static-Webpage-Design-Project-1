# 🌐 Decode Labs – Static Webpage Design Project 1

A modern and visually appealing static webpage developed using HTML5 and CSS3. This project focuses on creating a well-structured and attractive user interface while applying fundamental web design principles.

This project demonstrates the basics of front-end web development through a static webpage design. It showcases clean layouts, organized content sections, and responsive styling techniques to create an engaging user experience.
